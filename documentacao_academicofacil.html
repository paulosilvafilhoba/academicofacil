Interface IA acadêmica

academicofacil.com.br/agente-academico — 875 linhas · tema dark tech

Layout de 3 colunas
Sidebar esquerdaHistórico de conversas + toggle Modo Chat / Modo Acadêmico
Área centralTopbar com stage chip + barra de qualidade animada + chat + pills de ação + input
Painel direitoContexto do pedido + rubrica Nota 9+ por critério + riscos identificados

Dois modos de operação

💬 Modo Chat

Multi-turn conversacional. Histórico das últimas 8 mensagens enviado à API. Resposta em texto direto. Ideal para tirar dúvidas, revisões rápidas e orientações.

📚 Modo Acadêmico

Resposta em JSON estruturado com rubrica Nota 9+. Mostra score por critério, riscos identificados, pendências e próxima ação. Atualiza a barra de qualidade em tempo real.

Acesso

# Modo livre (qualquer usuário autenticado)
academicofacil.com.br/agente-academico

# Vinculado a um pedido específico
academicofacil.com.br/agente-academico?pedido=ORDER_ID
O acesso exige autenticação Firebase. Usuário não logado é redirecionado para /dashboard?redirect=ia-academica.
